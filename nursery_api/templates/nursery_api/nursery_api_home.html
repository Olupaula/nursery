Determine Admission Status

Introduction

This Api determines if a prospective student (child) should be admitted into a Nursery School using six features, namely:
  1. Kind of Parents ( i.e. parent = { "usual": 0, "pretentious": 1 , "greatly pretentious": 2} )
  2. Parent has Nursery Education ( i.e. has_nurs = { "proper": 0, "less proper": 1, "improper": 2, "critical": 3, "very critical": 4 } )
  3. Number of Children, ( i.e. children = { 1: 1, 2: 2, 3: 3, "more": 4} )
  4. Housing Status of the Child's Family ( i.e. housing = { "convenient": 0, "less convenient": 1, "inconvenient": 2} )
  5. Social Status ( i.e. social = { "non problematic": 0, "slightly problematic": 1, "problematic": 2} )
  6. Health Status of the Child ( i.e. health = { "priority: 0, "not recommended": 1, "recommended": 2} )

How to Use

Below is an example, in python file, of how to use this API: suppose there is a prospective student whose parents are pretentious, have less proper nursery education, have only one child, have an inconvenient housing, is slightly problematic, and the child's health is recommended as shown in the table 1.0 below, then this API can be used thus:

Table 1.0: Sample Data Point
Parents Nursery Children Housing Status Social Status Health Status
Pretentious Less proper nursery education 1 Inconvenient housing Slightly problematic Recommended

The API can be used in a python file (named client.py in this case) as shown below

client.py
            import requests

            endpoint = "http://127.0.0.1:8000/nursery_api"
            dict = {
                'parents': 1,
                'has_nurs': 2,
                'children': 1,
                'housing': 2,
                'finance': 1,
                'health': 2
            }

            predict = requests.post(endpoint, json=dict)
            print(predict)
